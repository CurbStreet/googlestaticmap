Class GoogleStaticMap
In: lib/googlestaticmap.rb
Parent: Object

Main class for creating a static map. Create an instance, Set attributes that describe properties of the map. Then call url to get a URL that you can use as the src of an img tag. You can also call get_map to actually download the map from Google, and optionally write it to a file.

Attributes

center  [RW]  MapLocation for the center of the map. If this is not specified, the map will zoom to the markers
format  [RW]  format of the image: 
 png8 - 8 bit PNG (default)
 png32 - 32 bit PNG
 gif
 jpg
 jpg-baseline - non-progressive JPEG
height  [RW]  Height of resulting image in pixels, defaults to 350, maximum 640
maptype  [RW]  Type of map to create: 
  roadmap (default)
  satellite
  terrain
  hybrid - satellite imagery with roads
markers  [RW]  An optional array of MapMarkers
paths  [RW]  An optional array of MapPaths and/or MapPolygons to draw
sensor  [RW]  Applications that determine the user‘s location via a sensor must set this to true, defaults to false
width  [RW]  Width of resulting image in pixels, defaults to 500, maximum 640
zoom  [RW]  0 (the whole world) to 21 (individual buildings), only set if center is set

Public Instance methods

get_map(output_file=nil)

Connects to Google, retrieves the map, and returns the bytes for the image. Optionally, pass it an output name and the contents will get written to this file name

relative_url()

Returns the URL to retrieve the map, relative to maps.google.com Example - "/maps/api/staticmap?params…"

url()

Returns the full URL to retrieve this static map. You can use this as the src for an img to display an image directly on a web page Example - "maps.google.com/maps/api/staticmap?params…"
